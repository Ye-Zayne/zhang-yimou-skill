# 05 · 关键创作决策与可迁移方法

Use this file when building a director treatment or when the script feels flat and needs a decisive style system.

## Decision 1：选一个视觉母题，不选一堆形容词

Bad:

- "高级、东方、宿命、浓烈、有张力。"

Better:

- "红色座次牌控制婚宴秩序；女主每次被羞辱，座次牌都离她更近；结尾她把空座次牌放到婆婆面前。"

Method:

1. Choose one motif.
2. Give it a first meaning.
3. Let power claim it.
4. Let the protagonist reclaim or destroy it.

## Decision 2：把社会规则拍成身体动作

Bad:

- "他们家很重男轻女。"

Better:

- "每次上菜，男丁先动筷；女主伸手夹菜时，全桌筷子同时停住。"

Method:

- Translate concept into table manners, queue order, seating, documents, keys, thresholds, phone placement.

## Decision 3：用程序制造现实主义压力

Bad:

- "医院不负责，女儿很愤怒。"

Better:

- "女儿每次说出父亲名字，窗口里的人只重复一句：系统里查不到。第三次，叫号屏突然跳出父亲的缴费记录。"

Method:

- Create a repeated official response.
- Let an object or system error break it.
- Keep performance restrained until the break.

## Decision 4：反转前先让观众相信错误秩序

Bad:

- "最后突然发现婆婆才是坏人。"

Better:

- "前半段所有仪式都证明儿媳不懂规矩；后半段发现那些规矩本来就是为了掩盖母亲死亡真相。"

Method:

1. Give the institution/family a plausible story.
2. Make the crowd enforce it.
3. Plant a contradictory object.
4. Let the object reassign guilt.

## Decision 5：大场面低预算化

Bad:

- "上百人围观，鼓乐齐鸣。"

Better:

- "楼道三层邻居同时探头，手机灯一盏一盏亮起；画外传来远处婚宴鼓点。"

Method:

- Replace quantity with pattern.
- Use offscreen sound to imply scale.
- Use vertical stacking, reflection, shadow, and phone screens.

## Decision 6：静态确认保护视频一致性

Before video prompt generation:

- Confirm character faces, costumes, age, key props.
- Confirm location geometry, entrances, light direction, color palette.
- Confirm 2x2 or 3x3 storyboard grid.
- Confirm each 15-second unit has one action and one camera strategy.

If not confirmed, stop and output a checklist. Do not continue to Seedance or batch video.

## Decision 7：把短剧爽点变成图像爽点

Common short-drama beats:

- 反杀
- 打脸
- 复仇
- 误会解除
- 身份曝光
- 金手指

Zhang-inspired visual translation:

- 反杀 = 被压在桌下的物件回到桌面中心。
- 打脸 = 全场队形从压迫主角变成围住加害者。
- 复仇 = 仪式继续，但主持人换了。
- 误会解除 = 被遮挡的门/窗/布被打开。
- 身份曝光 = 身份物件从伪装者身上转移到真实者手中。
- 金手指 = 不要凭空开挂，变成早已埋下的物件证据。
